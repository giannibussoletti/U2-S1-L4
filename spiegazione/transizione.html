Transizioni in css

una transizione rappresenta una applicazione di una o più regole CSS in maniera graduale e "rallentata". Una transizione si applica a seguito di un cambio di stato in CSS. Il più comune è il cambio di stato derivato da un evento di HOVER. Quando il mouse si sovrappone ad un elemento HTML.
